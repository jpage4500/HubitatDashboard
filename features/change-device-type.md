# Change Device Type

The app will automatically determine which device type every given device should default to. It does this by looking at the attributes of the device (ie: switch, power, lock, etc) as well as the device name. For example, a sensor that has "door" in the name will be set to the Door device type by default.

However, you can change the device type of any device:

* Long-click on the device and click on Edit
* Click on Device Type option
* Select the device type that you want to change this device to.&#x20;
* Note: the options you can switch to depend on what attributes your device supports
