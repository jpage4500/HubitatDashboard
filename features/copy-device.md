# Copy Device

You can make multiple copies of any hub device in the app. One reason to do this might be to show attributes of a device.

For example, this camera device tile has an 'armed' state and a 'temperature' state. If you want to show both on individual tiles, first make a copy of the device

![](<../.gitbook/assets/image (159).png>)

Next, change the device type of the "copy"

![](<../.gitbook/assets/image (106).png>)

The result is 2 device tiles that reflect both attributes on your dashboard

![](<../.gitbook/assets/image (183).png>)
