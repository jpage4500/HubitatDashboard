# Group Similar Devices

If you have several devices with the same (or similar) device type, there's a quick option to move them all into a folder.

* long-click on a device and click Edit
* click Group Similar Devices

![](<../.gitbook/assets/image (67).png>)

{% embed url="https://youtu.be/bC6lLsQ26AE" %}

If you want to remove all of the devices from the folder (move them back to the home screen), just long-click on the folder and click on "Delete folder"
