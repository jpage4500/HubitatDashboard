# Quick Settings Tiles

HD+ lets you add up to 2 quick settings tiles. These are buttons that show up in the quick settings panel.

<div align="left"><figure><img src="../.gitbook/assets/image (255).png" alt="" width="375"><figcaption></figcaption></figure></div>

## Configuring Quick Tiles

This varies by device manufacturer

* Start by opening the quick settings panel - by dragging down from the top of the screen.
* Click on the edit button (circled below)

<div align="left"><figure><img src="../.gitbook/assets/image (256).png" alt="" width="375"><figcaption></figcaption></figure></div>

* Find the HD+ quick settings tiles and drag them from the inactive to active sections

<div align="left"><figure><img src="../.gitbook/assets/image (257).png" alt="" width="375"><figcaption></figcaption></figure></div>

* Once you moved them to the active quick settings panel, either long-press or click on one of the tiles to enter the configuration screen

<div align="left"><figure><img src="../.gitbook/assets/image (258).png" alt="" width="375"><figcaption></figcaption></figure></div>

* Select the device to control - ideally a toggle-able device like a light or switch
* Give HD+ Background Location permission and Ignore Battery Optimizations in order for the tile (or widget) to stay up to date in the background.
  * Always on Background Location is needed for HD+ to get the currently connected WIFI SSID, which it uses to determine if local or cloud mode should be used
  * Ignore Battery Optimizations will prevent the OS from delaying or stopping the background refreshes
