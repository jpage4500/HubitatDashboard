# Set Timer

Some device types including light and switch will show an option in the details page to set a timer to turn the device on or off at a later time.

## Usage

* long-click on a device type such as a light
* click Set Timer
* select the state you want to change to (on/off) and when you want it to fire
* once set, you'll see the "Remove Timer" option - showing that it was successful.
