# Display Options

There are many global display options that can be changed (and more are added all the time)

![](<../.gitbook/assets/image (116).png>)

### Tile Size

Controls the tile size of a 1x1 (standard) sized device tile. Note that individual tiles can be made wider or taller (see [tile options](tile-options-icon-colors.md))

![](<../.gitbook/assets/image (81).png>)

### Device Name Size

This controls the size of the device name (label). Move the slider all the way to the left to hide the device name

![](<../.gitbook/assets/image (103).png>)

### Status Text Size

This controls the size of the tile status. Move the slider all the way to the left to hide the status

### Tile Padding

Controls space between tiles

### Icon Padding

Controls space around the main device icon

### Text Padding

Controls space around the main device text (ie: date/time)

### Rounded Corners

Controls how rounded the tile corners are

### Text Shadow

Uncheck to remove the drop-shadow used on main device text (date/time)

### Show Background Images

Uncheck to hide background images on tiles that also have text (weather, power tile)
