# Themes

There's several built-in themes. A theme defines the following items per device type (ie: doors, locks, lights, etc) + on/off state

* background color
* icon color
* text color

A theme does NOT currently change:

* icons

To change themes, open the navigation menu and hit Themes. Note that changing the themes will remove any customizations you currently have

![](<../.gitbook/assets/image (86).png>)

* The **Highlight Actionable** theme uses 1 background color for devices that can be toggled on/off (lights, locks) and another color for devices that don't have any 'action' associated with them (weather, web tiles, doors)
* The **Highlight ON/OFF** theme will set all 'on' devices to have 1 background color and all 'off' devices to have another background color. This is useful if you're interested in seeing if any lights are left on, for example.
