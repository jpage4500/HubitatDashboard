# Tile options (icon, colors)

In addition to changing the [theme](themes.md), just about every tile in the app can be individually customized.

To start:

* long-click on any tile to view Tile Options
* Click on Edit to view the edit screen

![](<../.gitbook/assets/image (82).png>)

For all of the below options, you can select where the customization should apply. Options are:

* This device (device name) - On / Off
  * Changes will only apply to **this device**
  * **On** = Change applies when device is On/Open/Unlocked
  * **Off** = Change applies when device is Off/Closed/Locked
* All (device type) - On / Off
  * Changes apply to **all devices that have this device type** (ie: doors)
  * **On** = Change applies when device is On/Open/Unlocked
  * **Off** = Change applies when device is Off/Closed/Locked
* All Devices - On / Off
  * Changes apply to **all devices** (for colors only)
  * **On** = Change applies when device is On/Open/Unlocked
  * **Off** = Change applies when device is Off/Closed/Locked

![](<../.gitbook/assets/image (126).png>)

### Change Icons

Select from one of the following image sources to browse icons

* **App Images** - built-in images
* **Hub Filemanager** - list all images in the Hub's filemanager ([http://hubitat.local/hub/fileManager](http://hubitat.local/hub/fileManager))
* **iconsdb.com** - search images from iconsdb.com
* **unsplash.com** - search images on unsplash.com (_best for dashboard backgrounds_)
* **Hubitat Device** - if you have any Hubitat devices which have an 'image' attribute such as Google Photos - you can use the image
* **Enter URL...** - enter a custom URL or remove/clear the existing URL
