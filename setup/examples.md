# Examples

Everyone has their own use case(s) for a smart home dashboard. Something I find useful is to see how others are using their dashboard and I'll try to re-create any ideas that look useful to my own setup.

Here's some examples of dashboards I use, with descriptions of what they try to do and where they're used.

***

**Device**: 10" Fire Tablet

**Location**: Main Entry Door&#x20;

**Purpose**: Time, [Weather](../tiles/weather-tiles/) (radar, temp, pollen forecast), family [calendar](../tiles/calendar.md), various security devices such as garage doors, locks, doors, lights. [Game schedules](../tiles/gametime-pro.md), [stock](../tiles/stock-tile.md) tile
