# Install HD+

There's several ways to get the app:

### Multi-platform version of HD+

#### [Apple App Store](https://apps.apple.com/us/app/hd-dashboard/id6759614539)

* Builds first to to [TestFlight](https://testflight.apple.com/join/Wdb6CEUh) so enroll there if you want the latest and greatest

#### [Desktop (Mac, Windows, Linux)](https://github.com/jpage4500/HubitatDashboard/releases)

* For desktop, I'm using an tool called JDeploy which handles building an installer for Mac/Windows/Linux and also auto-updates the app with no extra work!
* When installing, keep the box labeled 'update automatically' checked and each time you run the app it'll check for the latest and update

#### [Android Direct Download](https://github.com/jpage4500/HubitatDashboard/releases)

* There's an update option in the app (About -> Check for updates) which can download the update HD+

***

### Original Android-only version of HD+

#### [Google Play](https://play.google.com/store/apps/details?id=com.jpage4500.hubitat)&#x20;

* I'd recommend enrolling in the [beta](https://play.google.com/apps/testing/com.jpage4500.hubitat) version as well as it'll be updated frequently

#### [Amazon App Store](https://www.amazon.com/dp/B089N59PB9)

* Note: I don't update the Amazon App Store version as often as the Play Store.. if you want more frequent updates on an Amazon tablet or device, you might want to install the APK below and use the Check For Updates option built into the app

#### Android (original version) [Direct Download](https://jpage4500.s3.amazonaws.com/hubitat-dashboard/hubitat-dashboard-beta.apk)  ([version](https://jpage4500.s3.amazonaws.com/hubitat-dashboard/version-beta.txt) info)

* Make sure you disable automatic updates in the Play Store otherwise it'll replace the direct download version with the one from the Play Store
