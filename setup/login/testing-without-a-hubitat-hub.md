# Testing without a Hubitat Hub

HD+ requires a Hubitat Hub ([https://hubitat.com/](https://hubitat.com/)). However, if you want to test drive HD+ without a Hub you can.&#x20;

On the Login screen:

* Select the **MANUAL ENTRY** button
* Enter "**0.0.0.0**" for the IP Address field
* Select **LOGIN** button
