# Advanced Log

* HD+ only logs warnings and errors by default
* If you put the app into [debug mode](./#debug-mode), it'll log additional details
* Typically sending a [support log](./#sending-a-device-log) will be enough. However, there's a way to get more details in the log for harder to reproduce issues

## Advanced Logging

* Make sure the app is in Debug Mode
* To do this, open About -> Logs
* Click on Event Log filter (top-left) and select "Device Log (all)"
* Hit the SHARE button (top-right) and select an app to send the log \*or\* select the HD+ icon "Save to hub" to upload the log to the Hub's [filemanager](http://hubitat.local/hub/fileManager).
