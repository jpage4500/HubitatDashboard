# Calendar

![](<../.gitbook/assets/image (193).png>)![](<../.gitbook/assets/image (161).png>)

To add a calendar tile to the dashboard, first you need a calendar source in iCal format. Any source which supports iCal can be used including Google, Outlook, Apple.

For Google Calendar, you can find the iCal link by viewing the Calendar Settings page and looking for the "Public address in iCal format" URL or the "Secret address in iCal format". The URL should end with ".ical"

![](<../.gitbook/assets/image (131).png>)

Once you have an iCal URL, add a new Calendar tile to the dashboard and enter the URL

![](<../.gitbook/assets/image (76).png>)![](<../.gitbook/assets/image (79).png>)

* URL: the iCal URL
* Refresh Rate: how often to refresh/update the calendar
