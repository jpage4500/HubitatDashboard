# Safety Monitor (HSM)

This tile display the Safety Monitor (HSM) state of your Hub

### Configure HSM on your Hub

* HSM is available under Apps -> Add Built-In App
* See details [here](https://docs.hubitat.com/index.php?title=Hubitat_Safety_Monitor)

### Add HSM tile to the dashboard

* navigation menu -> Edit -> Add Tile -> HSM
* ![](<../.gitbook/assets/image (155).png>)
* The tile will automatically update to show the current HSM state

### Changing HSM state (arming / disarming)

* click on the HSM tile to see a dialog of arming options
* ![](<../.gitbook/assets/image (97).png>)
* click on any option to change the armed state
