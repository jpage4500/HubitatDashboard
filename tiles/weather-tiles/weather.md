# Weather

Hubitat Dashboard supports 2 different weather device drivers that can be added by Hubitat Hub - OpenWeatherMap-Alerts and the built-in Weather driver

## OpenWeatherMap-Alerts

The best one is OpenWeatherMap-Alerts - see the link below to install this to your Hub

{% embed url="https://community.hubitat.com/t/openweathermap-alerts-weather-driver/38249" %}

## Hubitat Dashboard Weather Device

* By default the weather device will display as a 1x1 tile - just showing the current temperature. However, you can resize the tile to show the forecast as well
* Clicking on the weather tile will show more information
